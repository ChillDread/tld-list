TLD List
========

Formats Available
-----------------

- Text
- JSON
- YAML
- XML
- HTML
- CSV
- SQL
    * MySQL
    * PostgreSQL
    * SQLite
- PHP

Build
-----

Currency list is available out of the box, but if you want to submit patches, add new formats,
update data source or contribute in any other way, you will probably want to rebuild the list:

```bash
$ composer install
$ ./bin/build
```
